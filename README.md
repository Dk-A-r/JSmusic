# Музыкальная гамма (C Major) на Web Audio API
**GitHub README**

---

## Описание

Этот небольшой веб-пример воспроизводит **мажорную гамму от "до" до "до" (C4 → C5)** с помощью **Web Audio API**.
Каждая нота звучит ровно **1 секунду**, последовательность длится **8 секунд**.

Используется чистый JavaScript без внешних библиотек.

---


## Как использовать

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/Dk-A-r/JSmusic.git
cd ./JSmusic

Откройте файл "index.html" в директории.
Нажмите кнопку "Play".

## Технические детали
<ul>
<li>Используется AudioContext и OscillatorNode</li>
<li>Каждая нота — отдельный осциллятор</li>
<li>Ноты запускаются последовательно с задержкой 1 секунда</li>
<li>Используется start(time) и stop(time) для точного контроля</li>
<li>Громкость: 1.0 (максимальная, без искажений)</li>
<li>После воспроизведения кнопка возвращается</li>
</ul>
